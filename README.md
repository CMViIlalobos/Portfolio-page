# Carlos Miguel Portfolio

Premium Laravel portfolio for Carlos Miguel S. Villalobos, designed for Laravel, Flutter, dashboard, workflow system, API, and full-stack development opportunities.

## Run locally

```bash
composer install
npm install
php artisan migrate --seed
npm run dev
php artisan serve
```

## Build

```bash
npm run build
php artisan test
```
